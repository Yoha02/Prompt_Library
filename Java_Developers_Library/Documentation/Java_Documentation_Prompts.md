## Swagger Doc Generation
### Use Case  
Generate a Swagger (OpenAPI) specification document for a given Java class.

### Prompt  
Generate a Swagger (OpenAPI) specification document for the provided Java class, extracting:  
- API endpoints  
- Request/response models  
- Applicable annotations  

Ensure the documentation includes metadata such as:  
- Title  
- Version  
- Descriptions  

Create a Swagger doc for the Java class provided in the context.
